>  参考导航 [team-nav](https://github.com/tuituidan/team-nav)、[菜鸟教程](https://www.runoob.com/)
>  保持极简的设计风格和可扩展性


## 技术栈

| 技术栈     | 框架说明                      |
|---------|---------------------------|
| 前端      | Vue3 + TS + ElementPlus   |
| 后端的技术栈  | SpringBoot + SpringDataJp |
| 数据库     | H2                        |
| Docker  | docker 部署                 |
| SaToken | 鉴权校验                      |
